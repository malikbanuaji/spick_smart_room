# spick_smart_room
features list:

Flask used for uploading presentation file and print

Monitoring:
  - Light intensity
  - Temperature
  - Humidity

Controling:
  - Curtain
  - Lamp
  - Printer
  - Presentation

Automation:
  - Auto Brightness (300 lux)
  - Alarm Lamp (get the configuration from input Google Assistant)
  
Hardware and others:
  - Raspberry Pi
  - Stepper + ULN2003
  - MOSFET
  - BH1750
  - DHT22/11
  - LED (3 Series + 1 100ohm resistor) 12v
  - HDMI Cable
